Technetium Contributor Guide
============================

Development Prerequisites
-------------------------
To develop Technetium, you'll need [.NET 7 SDK][dotnet] (or a later version).

Build the Project
-----------------
Execute the following shell command:

```console
$ dotnet build
```

Run Tests
---------
Execute the following shell command:
```console
$ dotnet test
```

[dotnet]: https://dot.net/
